> 2026-07-22, 인앱 브라우저로 실탐색하여 정리. UI/UX 참고용.
> 실제 열어본 화면: 뜨는 상권 / 나도 곧 사장(전체 플로우) / 분석리포트(간략, 이문1동×한식음식점) / 나는 사장 / 상권별 현황.

## 2. "나도 곧 사장" 플로우 (핵심 벤치마크)

```
진입 → 온보딩 질문: "창업하려는 업종이 있으세요?"
 ├─ [아니요, 고민중입니다] → 업종 선택 도우미
 │    지역(구) 선택 → 지표 토글: 점포수↑ / 매출↑ / 3년 생존률 / 폐업수
 │    → 업종 Top10 리스트 ("점포수 상승 업종" 등) → 업종 확정
 └─ [네, 있습니다] → 업종 먼저 확정
      대분류(전체/외식업/서비스업/소매업) → 세부 업종(외식업 10종 등)
      ↓
지역 선택 도우미: 지표 토글(점포수/매출/유동인구/주거인구)
  → "OO 상승 지역" Top10 (예: "동대문구 이문1동 · 136개 증가")
  → 리스트 클릭 = 지도 이동·마커 연동, 행정동 폴리곤+수치 라벨
  ↓
영역 클릭 → 팝업(상권유형·면적·위험도) → [분석리포트] 버튼
  → 간략 보고서 / 분기 종합 보고서 선택 → 사이드패널 리포트
  ↓
(선택) "창업 예정지 상세분석" — 보행권역 / 반경 / 다각형 그리기
```

특징:
- **온보딩을 '질문'으로 시작** — 기능 나열 대신 사용자 상황 분기. 두 갈래가 결국 같은 리포트로 수렴.
- 첫 방문 **튜토리얼 오버레이** (단계별 말풍선 + "다시 보지 않기").
- Top10 리스트 ↔ 지도 양방향 연동. "지도에 전체 지역 표시" 체크박스.
- 지도 레이어 토글 6종: 상권영역 / 점포이력 / 주변정보 / 유동인구(도로별) / 유동인구(건물별) / 인구밀도.

## 3. 분석리포트 (간략 보고서) 해부 — 이문1동 × 한식음식점 실측

**헤더**: 제목 + 기준분기 + 그래프 패턴 ON/OFF(접근성) + 새창/출력 + 문의전화
**탭**: 종합의견 | 업종분석 | 매출분석 | 인구분석 | 지역(배후지)분석
**면책·안내**: 선택면적/분석면적 표기, "추정정보이므로 실제와 다를 수 있음", 책임의 한계 안내 버튼, **서울신용보증재단 지원센터 안내(전화 + '위치로 지도 이동')**

### 종합의견 (리포트 최상단)
- 규칙 기반 불릿 4개: 점포수 증감→"입지선정 신중" / 매출 추세→"마케팅 강화" / 유동인구 추세→"경쟁 업소 출현 경계" / **자치구 내 순위 요약**
- KPI 3카드: 점포수 · 매출액(점포당 월평균) · 유동인구(3개월)
  - 각 카드: 현재값 + 전분기 대비 증감(±수치) + 전년 동분기 대비 증감 + **"나의 등수 8/14"** (자치구 내 행정동 순위)
- Best 요약 3종:
  - Best 업종: 점포수/개업/폐업/영업기간별 1위
  - Best 매출: 성별·연령/요일(토 17.3%)/시간대(17~21시)/최대 매출업종
  - Best 유동인구: 성별/연령(20대 17.1%)/요일(일)/시간대(00~06시)

### 섹션별 지표 (각 지표 = 동일 블록 패턴)
> **블록 패턴**: ① 한 줄 요약(핵심값 볼드) ② 전년 동분기/전분기 증감 배지 ③ 규칙 기반 조언 문장 ④ 추이 차트(분기별) ⑤ 서울시/자치구/행정동 3단 비교

- **업종분석**: 점포수(136개, 서울 58,744/동대문 2,313 비교, 프랜차이즈vs일반) · 신생기업 3년 생존율(31.25%, 추이) · 평균영업기간(2.6년 vs 자치구 2.8 vs 서울 2.8) · 개업(16개)/폐업(7개) 현황 · 업종분포(외식/서비스/소매 %)
- **매출분석**: 점포당 월평균 매출(1,454만원) · 매출건수(495건) · 요일별(토 최고→"주말 고객 마케팅") · 시간대별(17~21시) · 성별(남 60.3%→투자 조언) · 연령대별(업종 대분류별 각각, **데이터 없으면 "해당하는 데이터가 없습니다"**)
- **인구분석**: 유동인구(일평균 138,876명 + **밀도 120,227명/ha**) · 성별연령별(여성 20대) · 요일/시간대별 · 주거인구(17,724명, 170명/ha) · 직장인구(1,106명, 11명/ha) · 각각 성별연령별
- **지역(배후지)분석**: 가구세대(10,654가구, 아파트vs일반) · 아파트(면적대·가격대 분포) · 주요/집객시설(관공서·유통점·금융기관·영화공연·교통시설·학교·병원) · **소득수준(7분위 + 금액구간 + 자치구/서울 비교)** · 소비트렌드(카테고리별 %) · **임대시세(3.3㎡당 157,845원, 자치구 대비 -4,128원/서울 대비 +24,754원 + "고정비 관리 유의" + 표본부족 명시)**

### 문장 생성 방식 (우리와 동일한 규칙 기반)
- `[지표]는 [값]입니다` → `[증감 판단]` → `[행동 조언]` 3단 구성. 값은 볼드 강조.
- 조언은 조건부 템플릿: "주말 매출↑ → 주말·원거리 고객 마케팅", "폐업률↑ → 신중한 접근", "임대료 서울 대비↑ → 고정비 관리 유의".

## 4. 그 외 메뉴 요점

- **뜨는 상권**: [뜨는 동네(행정동) | 뜨는 상권] 토글 × 지표 4종 × 상세조건(분기/년, 전분기/동분기 비교, 업종 대분류, 유동인구는 연령대 필터). 지도에 행정동별 수치 라벨.
- **나는 사장**: 점포위치(주소 검색 or 지도 클릭) → 업종 → 분석영역(보행권역/반경/다각형) 3단계 + **"비교담기"(상권 비교 바스켓)**.
- **상권별 현황**: 지표(점포수/신생기업 생존율/연차별 생존율/평균영업기간/개폐업/인구/소득·가구) × 상권유형(골목/발달/전통시장/관광특구) × 분기 × 업종(생활밀접 ~100종) 검색 테이블.

## 5. 우리 web-demo 갭 분석 & 도입 후보

이미 우리가 갖춘 것(동등 이상): 두 갈래 진입(자리부터/업종부터), 상권 내 업종 랭킹(golmok에 없는 **상권 단위** 업종 추천), 섹션형 리포트+정직성 문구, AI 인스펙터(차별점), 목업 폴백.

| 우선순위 | 도입 후보 (golmok 패턴) | 적용 아이디어 |
|---|---|---|
| ★★★ | **"나의 등수" 비교 맥락** | 리포트 KPI에 "이 상권에서 이 업종 매출 상위 N% / 자치구 내 M위" — 이미 salesPercentile 보유, 표기만 추가 |
| ★★★ | **종합의견 불릿** | 현 narrative 한 문단 → 지표별 불릿 3~4개(값+판단+조언)로 구조화 |
| ★★☆ | **온보딩 질문형 진입** | 첫 화면에 "창업하려는 업종이 있으세요?" 카드 → 모드 토글을 자연어 질문으로 |
| ★★☆ | **Top10 리스트 ↔ 지도 연동** | 업종 랭킹 행 hover/클릭 시 지도 하이라이트 (우리는 단방향) |
| ★★☆ | 증감 배지 (전분기/전년 동분기) | 분기 시계열 이미 보유(2025Q1~2026Q1) — exports에 증감 필드 추가 |
| ★☆☆ | 튜토리얼 오버레이 | 데모 시연용 단계별 가이드 ("다시 보지 않기") |
| ★☆☆ | 추이 미니 차트 | 리포트 매출/점포수에 분기 추이 스파크라인 (recharts 보유) |
| 보류 | 임대시세/소득/집객시설, 반경·다각형 분석, 챗봇, 비교담기 | 데이터 소스 추가 필요 — 대회 범위 밖 |

## 6. 법적/윤리 참고
- 공공 서비스의 **화면 구성·플로우 아이디어**를 참고하는 것이며, 마크업·이미지·문구를 그대로 복제하지 않는다.
- golmok도 모든 수치에 "추정정보, 책임의 한계" 면책을 붙인다 — 우리의 정직성 원칙과 부합, 유지할 것.
